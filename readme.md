# My own nvim configuration build


# Installation
Navigate to .config
```bash
cd ~/.config
```

Clone the repo
```bash
git clone https://github.com/nicoripkip/NVIMconfig.git
```
Rename the repo folder to nvim
```bash
mv NVIMconfig nvim
```
